# Data

## Download Instructions

This project uses the **UCI Online Retail II** dataset.

1. Go to: https://archive.ics.uci.edu/dataset/502/online+retail+ii
2. Download `online_retail_II.xlsx`
3. Place it in `data/raw/online_retail_II.xlsx`

## Notes
- The dataset contains two sheets: `Year 2009-2010` and `Year 2010-2011`
- Raw data is **not committed** to this repository (see `.gitignore`)
- Processed data will be generated by running `notebooks/01_eda.ipynb`
